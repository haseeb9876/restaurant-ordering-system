# 🍕 Royal Pizza Palace — Restaurant Ordering System

A modern, production-style Restaurant Ordering System built with a scalable full-stack architecture.

This project is designed as a real-world startup-level food ordering platform inspired by applications like Foodpanda and Uber Eats.

It includes:

- Customer ordering experience
- Admin management dashboard
- Kitchen management panel
- Progressive Web App (PWA)
- Mobile-first responsive UI
- Secure authentication system
- Dynamic restaurant branding
- Real-time order workflow
- Professional UX/UI polish

⚙️ Local Development Setup
1. Clone Repository
git clone https://github.com/haseeb9876/restaurant-ordering-system.git

cd restaurant-ordering-system
2. Frontend Setup
cd client

npm install

npm run dev

Frontend runs on:

http://localhost:5173
3. Backend Setup
cd server

npm install

npm run dev

Backend runs on:

http://localhost:5000
🔑 Environment Variables

Create .env inside server/

DATABASE_URL=

JWT_SECRET=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=

EMAIL_USER=

EMAIL_PASS=
